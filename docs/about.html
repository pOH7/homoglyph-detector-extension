What are Homoglyphs?

Homoglyphs are characters that look visually identical or very similar but have different Unicode code points. They can be used in phishing attacks, domain spoofing, and other deceptive practices.

Example: The word "master" can be written as "maѕter" using a Cyrillic 'ѕ' instead of Latin 's'. To the human eye, they look identical, but computers treat them as completely different characters.

How This Extension Works

  • Automatically scans web pages for suspicious Unicode characters
  • Highlights characters that look like Latin letters but come from other scripts (Cyrillic, Greek, etc.)
  • Provides detailed information about each suspicious character
  • Monitors for mixed-script usage that might indicate deception

Features

  • Real-time Detection: Automatically scans pages as they load
  • Visual Highlighting: Suspicious characters are highlighted with yellow background and red underline
  • Detailed Tooltips: Hover over highlights to see Unicode information
  • Script Detection: Identifies Cyrillic, Greek, and other non-Latin scripts
  • Performance Optimized: Minimal impact on page loading

Privacy & Security

  • This extension only analyzes text content on web pages
  • No data is sent to external servers
  • All processing happens locally in your browser
  • Settings are stored locally using Chrome's storage API

Usage Tips

  • Pay attention to highlights on login pages and URLs
  • Be extra cautious with highlighted domain names
  • Use the manual scan feature for important documents
  • Disable temporarily if highlights interfere with reading
